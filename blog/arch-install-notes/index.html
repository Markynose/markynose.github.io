Arch Install Notes (2026 Edition)

 2026-03-20 ·  1 min read · #arch #linux #install #btrfs

My annotated Arch Linux install process — partitioning, btrfs subvolumes, systemd-boot, and a few gotchas.

These are my personal install notes, kept here so I can reproduce my setup without digging through the wiki every time.

🔗Assumptions

  • UEFI system
  • NVMe drive at /dev/nvme0n1
  • Btrfs with subvolumes for easy snapshotting

🔗Partition layout

nvme0n1p1   512M    EFI System Partition
nvme0n1p2   rest    Linux filesystem (btrfs)
# partition the disk
fdisk /dev/nvme0n1

# format
mkfs.fat -F32 /dev/nvme0n1p1
mkfs.btrfs -L arch /dev/nvme0n1p2

🔗Btrfs subvolumes

mount /dev/nvme0n1p2 /mnt

btrfs subvolume create /mnt/@
btrfs subvolume create /mnt/@home
btrfs subvolume create /mnt/@snapshots
btrfs subvolume create /mnt/@log

umount /mnt

# mount with options
mount -o noatime,compress=zstd,subvol=@ /dev/nvme0n1p2 /mnt
mkdir -p /mnt/{home,.snapshots,var/log,boot}
mount -o noatime,compress=zstd,subvol=@home /dev/nvme0n1p2 /mnt/home
mount -o noatime,compress=zstd,subvol=@log /dev/nvme0n1p2 /mnt/var/log
mount /dev/nvme0n1p1 /mnt/boot

🔗Base install

pacstrap -K /mnt base base-devel linux linux-firmware \
    btrfs-progs networkmanager neovim git zsh

🔗Bootloader (systemd-boot)

arch-chroot /mnt

bootctl install

# /boot/loader/loader.conf
cat > /boot/loader/loader.conf << 'EOF'
default arch.conf
timeout 3
console-mode max
editor no
EOF

# /boot/loader/entries/arch.conf
cat > /boot/loader/entries/arch.conf << 'EOF'
title   Arch Linux
linux   /vmlinuz-linux
initrd  /initramfs-linux.img
options root=LABEL=arch rootflags=subvol=@ rw quiet
EOF

🔗Gotchas

  • If snapper is used, the .snapshots directory must exist on the root subvolume before mounting — otherwise snapper creates it as a regular dir and the subvolume mount fails.

  • compress=zstd without a level defaults to level 3. Use compress=zstd:1 for speed or zstd:9 for cold storage.

  • Don't forget btrfs-progs in the initramfs hooks if you want fsck at boot.
