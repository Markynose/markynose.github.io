Hello, World

 2026-04-05 ·  1 min read · #meta #zola #linux

First post. Why I started this blog, what it'll be about, and why a static site on Codeberg Pages.

Every developer eventually writes a "hello world" post. Here's mine.

🔗Why a blog?

I've been meaning to start writing for a while. Not to build an audience or sell anything — just to have a place to dump notes, document setups, and occasionally think out loud.

The format is simple: things I find interesting, things I build, things I break and then fix.

🔗Why Zola?

Static sites are the right tool for personal blogs. No database, no moving parts, fast deploys, easy backups. Zola is written in Rust, has zero runtime dependencies, and builds in milliseconds.

# install on Arch
pacman -S zola

# build and serve locally
zola serve

🔗What to expect

Posts will mostly be about:

  • Linux system administration and configuration
  • Shell scripting and automation
  • Hardware tinkering
  • Software I find interesting

No ads, no tracking, no JavaScript required to read a post.

That's it. Let's get writing.
